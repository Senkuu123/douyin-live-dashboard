# v0.1.0：Windows桌面端MVP

首个可用版本完成了单直播间实时采集、MySQL持久化和桌面运营看板。该版本已经通过2小时持续采集验证，并完成弹幕、礼物、进场、点赞、关注、分享、粉丝团和在线人数等消息的真实直播间联调。

## 主要功能

- 输入房间号或粘贴直播间链接开始采集。
- 实时展示当前在线、进场、弹幕、点赞、礼物和独立用户指标。
- 在线趋势按完整分钟更新，显示最近15分钟并标记礼物高峰、弹幕问题激增和在线下降。
- 展示实时弹幕、高价值事件流、问题队列和行动提示。
- 提供发言榜、送礼榜和粉丝团榜。
- 支持最低用户等级全局筛选。
- 支持九维直播态势评分及自定义权重。
- 支持行动提示聚合和建议文案开关。
- 支持本次会话CSV导出和最近历史会话恢复。
- 事件写入MySQL，平台消息ID去重，用户ID使用SHA-256哈希。

## 本版修复

- 修复在线人数在房间累计人数和实时在线人数之间跳变的问题。
- 修复匿名会话中独立用户数长期显示为1的问题。
- 修复直播标题无法实时回写的问题。
- 修复礼物连击累计数量造成重复入库和0钻石记录的问题。
- 修复长时间采集后行动提示停止更新的问题。
- 修复在线下降提示按秒重复生成的问题。
- 修复趋势图持续重建造成的刷新闪烁。
- 修复同一分钟多个趋势标签重叠、事件点偏离曲线和终点重复提示问题。

## 趋势判定

- 礼物高峰：本分钟不少于5件礼物，并达到前3分钟均值的2倍。
- 弹幕问题激增：本分钟不少于3条问题弹幕，并达到前3分钟均值的2倍。
- 在线下降：较上一分钟减少至少20人，并且降幅达到5%。
- 同类型事件至少间隔2分钟。

- 125分钟持续采集，入库35,637条弹幕，平台消息ID无重复。
- 礼物登录态联调120秒，入库2,241条事件，其中631条礼物消息、289位送礼用户。
- 7个测试文件、24项自动化测试通过。
- TypeScript类型检查、数据库冒烟、Electron生产构建和桌面界面冒烟通过。
- Windows便携版隐藏启动12秒保持运行。

## 运行要求

- Windows 10或Windows 11，x64。
- MySQL 8.0。
- 当前用户目录中的`.my.cnf`数据库连接配置。
- 需要完整礼物数据时，可在EXE同目录通过`.env.local`配置本机登录态。

## 安装步骤

1. 下载便携版EXE。
2. 配置MySQL和`.my.cnf`。
3. 双击运行软件。
4. 输入公开直播间房间号并开始采集。

便携版未使用商业代码签名证书，Windows可能显示SmartScreen提示。

## 已知限制

- 抖音网页版非公开协议发生变化时，采集功能可能需要同步升级。
- 匿名连接是否包含礼物消息取决于直播间和平台风控。
- 当前问题识别为本地关键词分类。
- 同一时间只支持一个直播间。
- 当前构建未配置自定义应用图标和商业代码签名。
